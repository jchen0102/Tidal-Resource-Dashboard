# Tidal Compute Cluster Monitoring Dashboard

React + TypeScript + Vite + TailwindCSS (dark-mode-first dashboard skeleton).

## Getting started

Install dependencies:

```bash
npm install
```

Start dev server:

```bash
npm run dev
```

## Notes

- Tailwind is configured using Tailwind v4’s Vite plugin (`@tailwindcss/vite`).
- The dashboard layout lives in `src/App.tsx` (header + filters + 3-row grid placeholders).
